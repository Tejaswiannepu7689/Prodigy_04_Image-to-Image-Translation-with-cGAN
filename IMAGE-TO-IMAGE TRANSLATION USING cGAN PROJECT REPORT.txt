IMAGE-TO-IMAGE TRANSLATION USING CONDITIONAL GAN (cGAN)

Abstract

This project implements Image-to-Image Translation using Conditional Generative Adversarial Networks (cGANs). The system learns a mapping between input and output image domains and generates translated images while preserving important visual features. The project demonstrates the application of Generative AI in computer vision tasks.

Introduction

Image-to-Image Translation is a computer vision technique that transforms an image from one representation to another. Conditional GANs extend traditional GANs by conditioning the generation process on input images, enabling tasks such as image colorization, style transfer, and semantic image synthesis.

Objectives

- Perform image-to-image translation using cGANs.
- Learn mappings between image domains.
- Explore Generative AI applications in computer vision.
- Generate realistic translated images.

Technologies Used

Methodology

1. Data Collection

Input-output image pairs are collected for training.

2. Image Preprocessing

Images are resized, normalized, and prepared for model training.

3. Generator Network

The generator converts input images into translated output images.

4. Discriminator Network

The discriminator evaluates whether generated images are real or fake.

5. Training

Generator and discriminator are trained simultaneously using adversarial learning.

6. Image Translation

The trained model generates translated images from unseen inputs.
