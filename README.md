# Prodigy_04_Image-to-Image-Translation-with-cGAN
## Objective
Translate images from one domain to another using Conditional Generative Adversarial Networks (cGANs).

## Technologies Used
- Python
- PyTorch
- cGAN
- Torchvision
- Matplotlib

## Installation

pip install -r requirements.txt

## Run

python image_to_image_translation.py

## Features
- Image-to-image translation
- Image preprocessing
- Deep Learning implementation
- Visualization of results

## Applications
- Style Transfer
- Image Colorization
- Medical Imaging
- Satellite Image Processing
